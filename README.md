# An-lisis-de-Ventas-y-Desempe-o-Comercial-2023-2024-con-Power-BI-Python
Proyecto de análisis end-to-end donde se integran Power BI y Python para transformar datos en decisiones de negocio. Incluye limpieza de datos, EDA, visualizaciones ejecutivas, KPIs, análisis de cohortes y proyecciones con modelos predictivos.
